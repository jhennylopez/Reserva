MANUAL DE USUARIO

SISTEMA DE GESTIÓN DE RESERVAS RommyEC

Este manual proporciona las instrucciones necesarias para operar el sistema desarrollado por Luis Franklin Salazar Poma. El software utiliza una arquitectura de 4 capas para garantizar la integridad de los datos y una experiencia de usuario fluida.

1. Inicio de Sesión y Seguridad

El sistema cuenta con un control de acceso dual para proteger la información y delimitar funciones.

  • Identificación: Ingrese su correo electrónico y contraseña.
  • Roles de Usuario:
    • Administrador: Control total del sistema.
    • Huésped: Acceso limitado a sus reservas.
  • Cierre de Sesión: Siempre cierre sesión en equipos públicos.

2. Gestión de Huéspedes

2.1 Registro

  1. Ir a Huésped > Registrar.
  2. Ingresar cédula (10 dígitos).
  3. Completar datos personales.
  4. Presionar Guardar.

2.2 Actualización

Use la opción actualizar para modificar datos del cliente ingresando su cédula.

3. Catálogo de Alojamientos

  • Capacidad Máxima: Evita sobrecupos.
  • Descripción: Ej: “Cabaña con vista al volcán”.

4. Proceso de Reservaciones

4.1 Nueva Reserva

  1. Ir a Reservas > Nueva Reserva.
  2. Seleccionar alojamiento.
  3. Elegir fechas.
Nota: La fecha de salida no puede ser anterior a la de entrada.

  2. Confirmar registro.

4.2 Cancelación

Seleccione la reserva y presione eliminar con confirmación.
